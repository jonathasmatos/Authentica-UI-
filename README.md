# Authentica-UI

Frontend moderno desenvolvido com React/Vite para o ecossistema Authentica.

## 🚀 Tecnologias
- React 19
- Vite 7
- Framer Motion (Animações)
- Lucide React (Ícones)
- Axios (Conectividade)

## 🌐 Integração
O frontend está configurado para se conectar ao backend em produção no Railway:
`https://authenticaspringboot-production.up.railway.app`

## 🛠️ Como rodar localmente
1. Instale as dependências: `npm install`
2. Inicie o servidor de desenvolvimento: `npm run dev`
